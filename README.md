# CS210-Grocery-Tracking-Program

## Project Summary
For this project in CS 210, I developed a Grocery Tracking Program using C++. The purpose of this program was to track how often grocery items were purchased and display that information in a clear and organized way. The program reads data, processes it, and allows the user to view item frequencies and a histogram of purchases. This project helped me apply core programming concepts such as file handling, loops, conditionals, and functions to solve a real-world problem.

## What I Did Particularly Well
One thing I did particularly well in this project was organizing my code so it was easy to read and follow. I used clear variable names, proper indentation, and comments to explain what each part of the program does. I also made sure the program produced accurate results, especially when displaying the frequency of grocery items and the histogram output.

## Where I Could Enhance My Code
One area where I could improve my code is adding stronger input validation and error handling. For example, I could better handle situations where the file cannot be opened or if the user enters invalid menu options. I could also improve the structure by separating more functionality into reusable functions.

## Challenges and How I Overcame Them
The most challenging part of this project was working with file input and correctly tracking the frequency of each grocery item. I overcame this by testing my code step by step and checking the output frequently. I also used course materials and debugging to help solve problems.

## Transferable Skills
This project helped me develop skills such as problem-solving, debugging, file handling, and writing clean, readable code. These skills will transfer to future coursework and software development projects.

## Maintainability and Readability
I made this program maintainable and readable by using consistent formatting, meaningful variable names, and comments throughout the code. I also structured the program in a logical way so it is easy to follow and update in the future.
